Instructions for Uploading Datasets

Follow these steps to upload your own datasets:

  1. Access page: Visit the upload page through the dashboard by clicking on "Upload Data" in the header.
  2. Select file: Make sure your file is in CSV or JSON format. Click on the field next to "CSV or JSON File" and select the file from your computer.
  3. Enter keys: Fill in the following fields:
    • Timestamp Key: Enter the key for the timestamp.
    • Username Key: Enter the key for the username.
    • User ID Key: Enter the key for the user ID.
    • Post ID Key: Enter the key for the post ID.
    • Text Key: Enter the key for the text content.
  4. Upload file: Verify all the entered information for accuracy. Click the "Upload" button to start the upload process.
  5. Confirmation: Wait for a confirmation message. After a successful upload, an appropriate message will be displayed.
  6. Troubleshooting: If an error message is displayed, check the file and the entered keys for correctness and repeat the process.
