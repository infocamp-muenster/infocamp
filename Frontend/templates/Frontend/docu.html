Anleitung zum Hochladen von Datensätzen

Folgen Sie diesen Schritten, um Ihre eigenen Datensätze hochzuladen:

  1. Seite aufrufen: Besuchen Sie die Upload-Seite über das Dashboard, indem Sie auf "Upload Data" in der Kopfzeile klicken.
  2. Dateiauswahl: Stellen Sie sicher, dass Ihre Datei im CSV- oder JSON-Format vorliegt. Klicken Sie auf das Feld neben "CSV or JSON File" und wählen Sie die Datei von Ihrem Computer aus.
  3. Schlüssel eingeben: Füllen Sie die folgenden Felder aus:
    • Timestamp Key: Geben Sie den Schlüssel für den Zeitstempel ein.
    • Username Key: Geben Sie den Schlüssel für den Benutzernamen ein.
    • User ID Key: Geben Sie den Schlüssel für die Benutzer-ID ein.
    • Post ID Key: Geben Sie den Schlüssel für die Post-ID ein.
    • Text Key: Geben Sie den Schlüssel für den Textinhalt ein.
  4. Datei hochladen: Überprüfen Sie alle eingegebenen Informationen auf Richtigkeit. Klicken Sie auf die Schaltfläche „Upload“, um den Hochladevorgang zu starten.
  5. Bestätigung: Warten Sie auf eine Bestätigungsmeldung. Nach erfolgreichem Hochladen wird eine entsprechende Nachricht angezeigt.
  6. Fehlerbehebung: Falls eine Fehlermeldung angezeigt wird, überprüfen Sie die Datei und die eingegebenen Schlüssel auf Korrektheit und wiederholen Sie den Vorgang.
